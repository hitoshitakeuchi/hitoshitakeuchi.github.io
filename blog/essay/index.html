Poem

There are three dimensions

Ｘ, Y and Z

Like a game engine

Then later you hear there’s a fourth dimension

Quaternion

Like a game engine

Then some say there can be five, six, seven…

I knock off work

Between mathmatics and emotion

Like a rolling stone
